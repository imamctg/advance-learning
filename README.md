# Learning Management System (LMS)

A full-stack Learning Management System (LMS) built using **Next.js, MERN Stack, TypeScript, and Redux**. This platform allows instructors to create and manage courses while students can enroll, track progress, and take lessons.

## 🌐 Live Demo
https://advance-learning-client.vercel.app/en

## 🛠️ Technology Stack
- **Frontend:** Next.js, React, Redux, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express.js, MongoDB
- **Authentication:** JWT
- **Deployment:** Vercel (frontend), Render (backend)

## 🔹 Features
- Instructor & Student Authentication
- Course creation, update, and management
- Interactive dashboard for admin, students, and instructors
- Responsive design for desktop and mobile
- Secure API endpoints and data validation
